主要是JPAQUERY的实现方式：假定通过id字段关联。

第1种方法在JPAQUERY没法简单实现: 

``` sql
select *
from A
where id not in (select id from B)
```  

第2种方法JPAQUERY3.0以上的版本才支持.on: 

``` sql 
select A.*
from A LEFT JOIN B ON A.id = B.ID
WHERE B.ID IS NULL
```
